Features that do require login.


The following features do require the user to login.
  1. Adding a movie to favorites.
  2. Writing a movie review.
  3. Accessing Profile Page.

Feature 1: Adding a movie to favorites.


As shown below, a add to favorites button is provided when the user searches for a movie.

Feature 2: Writing a movie review.


As shown below, a small dialogue box, is displayed when the user clicks on the write a review button as shown above. For the dialogue box to appear the user has to be logged in.

Feature 3: Accessing Profile Page.


A user can view the reviews he has written and also view his favorite movies once they are logged in. In the image below, the user kartik1990 who is logged in, is viewing his profile, we will first see how his favorite movie looks like
