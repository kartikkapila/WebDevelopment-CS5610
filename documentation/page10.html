Overview



Movie Search Query Engine

Introduction

This project is about, searching for your favorite movies, viewing your friends favorite movies, their reviews and also adding your reviews at the same time. At the landing page, you can view the upcoming movies, you can browse over all the upcoming movies using an image slider. Also at the landing page, you can view the in theater movies.
Once you search for a movie, all the information about the movie, from its runtime to a youtube trailer is displayed. Also at this page, you can view the comments of registered users. You can visit their profile page without requiring to login. If you wish to leave a review or add the movie you searched to your list of favorites, you will need to login.

Features

The list of features are as follows:
  • Its a single page application.
  • Retrieves Data from External Data Providers using Web Services.
  • Uses LINQ to SQL to query the database tables.
  • Has a Profile Page for each registered user.
  • Uses Module Pattern to organize javascript files.
  • Uses templates to separate jquery and css code.
