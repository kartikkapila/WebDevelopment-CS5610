Profile Page


The Profile Page consists of the following sections.
  1. Welcome Area
  2. User Details
  3. Read Reviews/Check Favorites Option

Section 1: Welcome Area


The welcome area, holds a message which can be of three types:
  1. Welcome "username" --> The user is logged in and is viewing his profile.
  2. You are currently viewing ... profile --> A guest user is viewing a profile.
  3. Welcome "username", You are currently viewing ... profile --> The user is logged in and is viewing someone elses profile.

Section 2: User Details


The user details that are displayed are
  • Profile Pic
  • Member Since Information
  • Profile Views
  • Email
  • Real Name
  • About Me
