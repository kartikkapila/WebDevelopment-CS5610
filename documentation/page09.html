Browser Support

This site works best on Mozilla Firfox. Although it works well on chrome, there is a small issue with the youtube video, which might look like a bug to you, but that is how it works on www.youtube.com as well. You can test it on chrome as follows:
  1. Go on www.youtube.com
  2. Search for "The Lion King"
  3. Click "30 minutes movies: The Lion King"
  4. Now search for any other thing, you will notice that the video is still playing in the background.
