Results Page


The Results Page consists of the following sections.
  1. Searched Movie Information
  2. YouTube Video
  3. Write Review/Add to Favorites Option
  4. Similar Movies
  5. Read Reviews

Section 1: Searched Movie Information


When a movie is searched for, the following information related to the movie is displayed:
  • Movie Image
  • Title
  • Genre
  • Synopsis
  • Tagline

Section 2: YouTube Video


As you can see from the image, above when the user searches for a movie, a trailer of that movie is displayed using the YouTube Player API.

Section 3: Write Review/Add to Favorites Option


Also the user is given the option to add the movie to the list of his favorite movies. He may also leave a review regarding the movie. For security reasons, both of these actions, require the user to login.

Section 4: Similar Movies


The user is shown a list of similar movies, based on the movie he has searched for. He can visit and read about those movies by clicking on the images of the similar movies.
You can check out my experiment on webservices to find out how I get the similar movies here.
