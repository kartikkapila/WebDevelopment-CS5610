Features that do not require login.


Note: Majority of the website can be accessed without requiring to login.
The following features do not require the user to login.
  1. Searching for a movie.
  2. Reading comments made by other users.
  3. Viewing Profile Pages of other users.

Feature 1: Searching for a movie


As shown below, a search bar is provided on the landing page, for the users to search for their favorite movies.
