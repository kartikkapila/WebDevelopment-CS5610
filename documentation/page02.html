Home Page


The Home Page consists of the following sections.
  1. Search Area
  2. Upcoming Movies Area
  3. In Theater Movies

Section 1: Search area


The search area is quite simple. It has a input field where the user can enter the movie he wishes to search for and then press the search button. The user may also login, using the login button.

Section 2: Upcoming Movies Area


As shown in the image below, the upcoming movie area, consists of a image slider. This slider can be used to navigate between all the latest upcoming movies, using the back and next button. When the user navigates, the information related to the movie changes. This information consists of:
  • Movie Title
  • Release Date
  • Synopsis
  • Runtime
  • Casts
You can find out how I find the upcoming movies, by visiting my experiment on that topic here

Section 3: In Theater Movies.


The user can also check out the in theater movies. To find more information about the movie, the user can click on the thumbnail, and the movie will be searched for.
